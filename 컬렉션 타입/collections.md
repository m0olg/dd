# 컬렉션 타입 (기본)
| 타입 | 설명 |
|:-----|:----:|
| `Array` | 순서가 있는 리스트 컬렉션 | 
| `Dictionart` | `키 `와 `값`의 쌍으로 이루어진 컬렉션 | 
|  `set` | 순서가 없고, 멤버가 유일한 컬렉션 |

---

### `Array` 배열
멤버가 순서를 가진 리스트 형태의 컬렉션 타입
```swift
// 좋아하는 것들을 순서대로 담기
var favorites = ["치이카와", "우사기", "딸기라떼"]

print(favorites[0]) // 치이카와 (순서로 접근 가능)

// 같은 것도 넣을 수 있음
favorites.append("치이카와") // 중복 가능
```

### `Dictionart` 딕셔너리
딕셔너리는 순서가 없고, 키와 값의 쌍으로 데이터를 저장, 각 키는 고유해야 하며 이를 통해 값에 접근 가능!!
```swift
// 캐릭터와 좋아하는 음료 연결
var drink = [
    "치이카와": "딸기라떼",
    "우사기": "딸기라떼"
]

print(drink["치이카와"]) // == 딸기라떼

// 키로 값 찾기


```

### `set` 세트
순서가 없고, 멤버가 유일한 컬렉션
```swift
// 좋아하는 것들 (중복 제거)
var likes: Set = ["치이카와", "우사기", "딸기라떼", "치이카와"]

print(likes) // 치이카와는 한 번만 들어감

// 순서 없음, 중복 자동 제거
```
---

## → 요약
- **배열(Array)** : 순서가 있는 동일 타입의 값 저장, 인덱스를 통해 접근 가능 ⭕️'
- **세트 (Set)** : 순서가 없고 중복이 없는 값, 저장,
- **딕셔너리 (Dictionary)** : 키-값 쌍으로 데이터 저장, 키를 통해 값 접근
