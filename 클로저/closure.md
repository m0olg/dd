### 클로저 기본

# 01 클로저란?
→ **클로저(Closure)는 코드에서 일급 객체로 취급되는 함수의 일종으로 코드 블록을 캡처하고 전달할 수 있지만, 이름이 없고 주변의 변수와 상수를 캡처할 수 있다는 점에서 함수와 구분** \
클로저는 보통 함수나 메서드의 인수로 전달되어 나중에 실행될 코드 블록을 정의하는 데 사용

# 02 클로저의 사용
**클로저의 기본 형태 ⬇️⬇️**
```swift
{ (매개변수 목록) -> 반환타입 in
    실행 코드
}
```

**<클로저 기본 예시>**
```swift
let sayHello = {
    print("쿼카에게인사해요안녕🐔")
}

sayHello() // 실행
```

### 함수의 전달인자로서 클로저
```swift
func doSomething(action: () -> Void) {
    print("시작")
    action()
    print("끝")
}

// 클로저 전달
doSomething {
    print("쿼카가 딸기라떼 마시는 중!")
}
/*
== 시작
   쿼카가 딸기라떼 마시는 중!
   끝
*/
```
클로저를 함수에 전달해서 원하는 동작을 나중에 실행 ⭕

# 03 다양한 클로저 표현
- 함수의 매개변수 마지막으로 전달되는 클로저는 `후행클로저(trailing closure)`로 *함수 밖에 구현할 수 없음*
- 컴파일러가 클로저의 타입을 유추할 수 있는 경우 매개변수, 반환 타입을 생략 가능
- 반환 값이 있는 경우 암시적으로 클로저의 맨 마지막 줄은 `return` 키워드를 생략하더라도 반환 값으로 취급
- 전달 인자의 이름이 굳이 필요없고, 컴파일러가 타입을 유추할 수 있는 경우 축약된 전달인자 이름 ($0, $1, $2 ...)를 사용 가능
